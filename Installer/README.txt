Este README tiene como objetivo explicar el proceso para actualizar el Macro Tracker cuando haya nuevas versiones disponibles y de desee instalarlas. Si se desea saber qué es el Macro Tracker, cómo funciona o qué funcionalidades ofrece por favor diríjase al README.md del repositorio en [https://github.com/Biosbardos/macro-tracker]. Allí también estará disponible el código si desea consultarlo.


CÓMO ACTUALIZAR EL MACRO TRACKER

Si desea actualizar su Macro Tracker por una versión más moderna deberá:

-Ir al repositorio del proyecto en GitHub [https://github.com/Biosbardos/macro-tracker] y descargar el installer de la última Release (en la izquierda del todo de la página justo debajo de todos los iconitos de Readme, GPL-3.0 license, Activity, stars, watching, forks…).

-Ejecutar el instalador (no se preocupe porque el objetos.txt donde tiene sus comidas personalizadas no se sobrescribirá).

Esto es todo, espero que le haya sido útil y disfrute de la aplicación.
