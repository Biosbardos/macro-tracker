Descripción detallada

Estructura que define un usuario con sus datos nutricionales.

Incluye nombre, altura, peso, edad, fórmula elegida, calorías de mantenimiento, calorías restantes, metas de proteína, grasa y carbohidratos, y los valores restantes.
